🚀 Sobre o projeto 

Podcastr é uma aplicação Front End web construída no evento NLW 5, organizado pela Rocketseat.

A aplicação consiste em selecionar os podcasts para serem ouvidos e sempre manter na página inicial os 
últimos lançamentos.
